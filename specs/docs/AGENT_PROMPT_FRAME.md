## Structural elements that are load-bearing

1. **Section headers**: `-- N.N: NAME` — numbered, dotted, named.
   Sections are addressable state. Re-projecting the prompt = walking
   sections, expanding/collapsing under budget.

2. **Expandable tokens**: `[[ N.N.N : signature ]]` — deferred content
   the agent can *request* via tool call. Compression primitive.

3. **Continuation tokens**: `<$ N.N.N next |` and `<$ N.N.N |` —
   suspension points inside type declarations. The agent can resume
   rendering a type from a specific expansion point.

4. **Lock declarations**: `ORG/path/HEAD from (T1 - T2) (version: vN)` —
   the partition + time window + version the agent holds. Agent dies
   when lock expires. Other agents see the range as locked.

5. **Timing declaration**: round-trip estimate INCLUDED in the prompt.
   "Plan accordingly!" Agents know how long they have.

6. **Agent identity**: `agent-3235423` — stable identifier. "11'th
   MOMENT" — the agent is durable across invocations. Each moment =
   one execution window.

7. **Tool pipeline as expand chain**: `tools = expand(tools[-1], agent/vN/output/expand, [['1.1.5.1']])` —
   the tools available to the agent are themselves the result of
   previously-requested expansions. The tool set is a projection of
   prior outputs onto prior tool templates.

8. **Value preamble**: The VALUES section. Not filler. The agent's
   principles are part of the prompt and influence inference.

9. **"Respond with merge into lattice"**: `1.4: RESPONSE` — the agent's
   output is a *merge proposal* that either applies or rejects based
   on type coherence with the partition it owns.

## Where this is being built

Not yet, fully. The substrate can project state like this (readers,
gaps, suspended blocks, admission). The *rendering pipeline* — take
a Sequence + agent identity + budget → produce THIS shape of prompt —
is not implemented. Closest existing piece:
`services/contextgraph/src/agent-loop.ts` (landed by
`project_agent_loop_landed`) has a state→LLM→mount loop but renders
a much simpler prompt shape.
